# csk-terminal

## 布局

![panel](./docs/panel.png)

右上角红框为操作栏，篮框为视图交互栏。

操作栏按钮：
- 清除视图内容
- 连接/断连 串口
- 保存视图所有内容
- 视图窗口最大化
- 关闭

## 使用

* 打开 `panel` 视图，并点击 `CSK TERMINAL`.
* 点击对应操作按钮
